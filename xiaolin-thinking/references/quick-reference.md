## 🔰 快速学习四步法

```
① 收集 → ② 搭框架 → ③ 追问Why → ④ 输出检验
```

| 步骤 | 关键动作 | 时间分配 | 常见错误 |
|------|---------|---------|---------|
| ① 收集 | 搜索引擎 + AI + 书籍目录 + 综述 | 20% | 一上来就钻细节 |
| ② 搭框架 | 思维导图 + 参考前人 + 不断提问 | 30% | 跳过这步直接啃知识点 |
| ③ 追问 | 问Why + 时间线 + 流程化 | 30% | 被别人的框架牵着走 |
| ④ 输出 | 脑图/讲给人听/写文章 | 20% | 以为"看懂了"就是"会了" |

**终极检验**："合起书，能用自己的话串起来吗？"

## 📌 五问拆题法

拿到一个问题，依次回答：

```
① 这事怎么开始的？      → 找历史起点
② 为什么变成现在这样？    → 画因果链
③ 跟普通人有什么关系？    → 建立连接
④ 有什么反直觉的点？      → 找钩子
⑤ 能用什么比喻秒懂？      → 翻译准备
```

## 🔍 六大思维工具

| # | 工具 | 一句话操作 |
|---|------|-----------|
| 1 | 🔍 历史纵深 | 时间轴拉长50-100年，从"怎么开始的"切入 |
| 2 | ⛓️ 因果链条 | 画出A→B→C→D完整多米诺，每步回答"为什么" |
| 3 | 🔄 跨案例比较 | 找不同国家/时代的类似案例，对比找规律 |
| 4 | 🎯 多维归因 | 至少从经济/政策/心理/制度/文化/历史六个维度分析 |
| 5 | 🔬 制度分析 | 关注"游戏规则"本身，而非只看玩家行为 |
| 6 | 🪞 翻译者思维 | 先找底层逻辑，再找日常类比 |

## 🎬 六幕叙事结构

```
🎣 钩子 → 📖 溯源 → 🔗 链条 → 🎭 高潮 → 💡 拆解 → 🔭 总结
```

| 幕 | 目标 | 关键技法 |
|---|------|---------|
| 🎣 钩子 | 3秒抓住人 | 灾难故事 / 反直觉数据 / 悬念提问 |
| 📖 溯源 | 建时间线 | 回到足够远的起点，每个节点变故事 |
| 🔗 链条 | 讲因果 | A→B→C每步解释"为什么"，穿插比喻 |
| 🎭 高潮 | 拉张力 | 数据变画面，描述人的反应 |
| 💡 拆解 | 教概念 | 微缩模型、对比表格、先直觉后公式 |
| 🔭 总结 | 回答"跟我有什么关系" | 个案→普遍规律→实操框架 |

## 🧋 七大比喻模型

| 比喻 | 用来解释 |
|------|---------|
| 🧋 奶茶店 | 供需、定价、经营决策 |
| 🌤️ 天气预报 | 预测、概率、不确定性 |
| 🏠 买房/租房 | 资产、负债、杠杆 |
| 🛡️ 保险 | 对冲、风险转移 |
| 🎪 演唱会 | 供需定价、时间价值 |
| 🏥 看病/体检 | 诊断、指标、风险评估 |
| 🎰 赌场 | 赔率、风险收益比 |

## 📐 信息筛选三原则

```
✅ 找第一手信息，不看二手解读
✅ 看结果性数据，过滤情绪噪音
✅ 等一个月再判断，过滤短期炒作
```

## ⚡ 三个"反常识"心法

1. **不当爱因斯坦**：站在巨人肩膀上，精力放在最上面那薄薄的创造层
2. **不陷细节兔子洞**：先整体后局部，先框架后知识点
3. **框架 > 知识点**：问专家问框架性问题，细节自己搜

## 🚫 表达禁区

```
❌ "XX是指……"（教科书开头）
❌ "基于多因子分析框架"（术语堆砌）
❌ "背后的神秘力量"（阴谋论）
❌ "明天大概率会涨"（股市预测）
❌ "再不学就来不及了"（贩卖焦虑）
❌ "众所周知"（冗余客套）
```
