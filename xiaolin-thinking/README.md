# 🧠 xiaolin-thinking

> 小Lin思考与分析方法论引擎——像小Lin一样学习、思考、分析和表达。

## 这是什么

这是一个基于小Lin说（北大金融+华尔街背景知识博主）的**思维方法论Skill**。它不是教你写文章，而是教你**如何思考**。

小Lin每期视频都涉及不同专业领域——从日本经济泡沫到期权波动率，从硅谷银行暴雷到全球芯片产业链。她不可能每个领域都是科班出身，但她有一套方法论，能在短时间内把陌生领域吃透到可以给几百万人讲明白的程度。

**这个Skill把她的方法论提炼成了可操作的SOP。**

## 核心能力

| 场景 | 说法 | 得到什么 |
|------|------|---------|
| 🔰 **快速学习** | "帮我快速了解区块链" | 知识框架 + 学习路径 + 关键问题清单 |
| 🔍 **深度分析** | "帮我分析为什么硅谷银行倒了" | 因果链拆解 + 多维归因 + 核心洞察 |
| 🗣️ **通俗转化** | "怎么给外行讲清楚期权" | 比喻翻译 + 故事框架 + 表达方案 |

## 方法论架构

```
快速学习四步法（输入）
  收集 → 框架 → 追问Why → 输出
        ↓
五问拆题法（转化）
  起源 → 因果 → 连接 → 反直觉 → 比喻
        ↓
六幕戏剧法（输出）
  钩子 → 溯源 → 链条 → 高潮 → 拆解 → 总结
```

## 与 xiaolin-style 的区别

| | xiaolin-thinking（本Skill） | xiaolin-style（创作引擎） |
|---|---|---|
| **定位** | 思维方法论引擎 | 内容创作引擎 |
| **核心问题** | "如何思考和学习" | "如何写出好文章" |
| **输出** | 知识框架、分析报告、比喻翻译 | 小Lin风格HTML长文 |
| **重点** | 快速学习四步法、五问拆题法、六大思维工具 | 六幕叙事结构、HTML模板、视觉设计 |
| **关系** | 上游（先想清楚） | 下游（再写出来） |

简单说：**先用 xiaolin-thinking 想清楚，再用 xiaolin-style 写出来。**
