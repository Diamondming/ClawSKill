# QClaw Skill: 投研资讯还原与扩展分析

这个 skill 文件是：`cls-article-reconstructor.md`

## 用途

用户输入一个财联社、金十数据或其他资讯链接后，Skill 会引导 QClaw：

- 先尝试读取公开页面
- 判断是否能直接获取全文
- 若遇到付费墙或登录限制，则基于公开信息还原文章逻辑
- 将完整文章直接输出在对话中
- 再按投研框架补充分类、题材梳理、个股映射、产业链扩展、交易逻辑和跟踪指标
- 如果用户额外要求，再保存为本地 Markdown 文件

## 更像投研助手的输出

默认不只输出“文章还原”，还会继续给出：

- 核心投研结论
- 题材分类
- 受益标的梳理
- 产业链或同主题扩展
- 后续跟踪指标
- 风险提示

## 安装方式

将以下文件复制到 QClaw/OpenClaw 的 skills 目录：

- `cls-article-reconstructor.md`

如果你的 QClaw 支持从 GitHub 安装，也可以把这个仓库作为单 skill 仓库导入。

## 使用示例

```text
帮我根据这个链接还原文章：
https://www.cls.cn/detail/2321939
```

```text
分析这个金十链接，无法读取全文就用公开信息还原，直接输出全文
```

```text
分析这个财联社链接，并按题材分类、受益股和产业链扩展一起输出
```

```text
根据这个链接还原内容，再拆成：文章类型、核心逻辑、直接受益标的、风险点
```

```text
按投研框架分析这篇文章，除了还原内容，再补充交易逻辑、映射标的和后续验证指标
```

## 默认输出

- 对话中的完整文章
- 题材分类
- 个股映射或产业链扩展
- 交易逻辑
- 跟踪指标
- 风险提示
- 如用户要求再保存，则可输出：`reconstructed_cls_<id>.md`
- 或 `reconstructed_article_<timestamp>.md`

## 设计边界

- 只做合规还原，不绕过付费墙
- 不伪造付费原文
- 明确区分公开事实与推断内容
